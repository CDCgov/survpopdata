# Join named population rows to district-year shapes

Join named population rows to district-year shapes

## Usage

``` r
join_pop_to_district_year_shapes(pop_named, district_long)
```

## Arguments

- pop_named:

  Tibble with Admin names, year, age columns, datasource.

- district_long:

  District-year long shapes table.

## Value

Tibble joined to year-valid district shapes.
