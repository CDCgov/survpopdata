# Load Pakistan district population patch

Load Pakistan district population patch

## Usage

``` r
load_pakistan_patch(
  pakistan_file_path =
    "GID/PEB/SIR/Data/pop/pop raw/csv files/2022_2023 Population Pakistan.csv",
  edav = TRUE
)
```

## Arguments

- pakistan_file_path:

  Path to Pakistan patch file.

## Value

Tibble with Admin names, year, age columns, datasource.
