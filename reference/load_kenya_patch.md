# Load Kenya district population patch

Load Kenya district population patch

## Usage

``` r
load_kenya_patch(
  kenya_file_path = "GID/PEB/SIR/Data/pop/pop raw/csv files/Kenya_SubCounty_pop_2018.csv",
  edav = TRUE
)
```

## Arguments

- kenya_file_path:

  Path to Kenya 2018 file.

## Value

Tibble with Admin names, year, age columns, datasource.
