# Indonesia province patch

Indonesia province patch

## Usage

``` r
load_indonesia_patch(
  indonesia_file_path =
    "GID/PEB/SIR/Data/pop/pop raw/csv files/Papua Population u15 2023.csv",
  edav = TRUE
)
```

## Arguments

- indonesia_file_path:

  `str` File path to the Indonesia patch for 2023.

- edav:

  `logical` Whether the file is in EDAV.

## Value

`tibble` Indonesia patch
